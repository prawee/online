### (1) Adaptive Learning Systems — Chen, Nian-Shing Chen, Sabine Graf, Gwo-Jen Hwang

ผู้วิจัย
 - Nian-Shing Chen
 - Sabine Graf
 - Gwo-Jen Hwang

หัวข้อเกี่ยวกับอะไร
 - สำรวจระบบการเรียนรู้แบบปรับตัว (Adaptive Learning Systems)
 - เน้นที่วิธี adaptivity ตามลักษณะเฉพาะของผู้เรียน ได้แก่ learning styles, cognitive abilities, affective states (อารมณ์/ความรู้สึก), context/situation

ใช้เทคนิคอะไร
 - เทคโนโลยี adaptive hypermedia systems, personalized learning ผ่าน learner profiles ที่เก็บข้อมูลลักษณะผู้เรียนต่าง ๆ เช่น ความสามารถทางสติปัญญา, สไตล์การเรียนรู้, อารมณ์ และบริบทของผู้เรียน
 - มีการปรับเนื้อหาหรือเส้นทางการเรียน การนำเสนอเนื้อหา (sequence, type) ตามลักษณะผู้เรียนเหล่านี้

ผลการวิจัย
 - พบว่า adaptivity ตาม learning styles ช่วยลดเวลาเรียนของผู้เรียนและเพิ่มความพึงพอใจของผู้เรียน
 - มีศักยภาพสูงในการเปลี่ยนวิธีการศึกษาแบบครูเป็นศูนย์กลาง ไปสู่ผู้เรียนเป็นศูนย์กลาง (learner‐centred) โดยเฉพาะในสภาพแวดล้อมออนไลน์และทุกที่ทุกเวลา
 - อย่างไรก็ตามยังมีเรื่องท้าทาย เช่น การนำไปรวมกับระบบ LMS เดิม, ความซับซ้อนของการออกแบบระบบ, ความเข้าใจเกี่ยวกับผู้เรียน, ความสามารถในการปรับปรุงตามสภาพอารมณ์หรือสถานการณ์จริง ๆ และข้อกำหนดโครงสร้างพื้นฐาน

ความคิดเห็นต่ออนาคต
 - มีแนวโน้มที่ adaptive learning จะมาแรงมากขึ้น โดยเฉพาะในโลกการศึกษาที่เน้นการเรียนออนไลน์ / แบบผสม (blended / hybrid) เพราะผู้เรียนต้องการความยืดหยุ่นและเหมาะสมกับตนเองมากขึ้น
 - ต้องมีการวิจัยเพิ่มเติมในเรื่องการประเมินอารมณ์ (affective states) ของผู้เรียน, พฤติกรรมในบริบทจริง, และการออกแบบที่ทำให้ระบบปรับแบบ real-time ได้ดียิ่งขึ้น
 - โครงสร้างพื้นฐาน (network, data, privacy, usability) และการยอมรับจากผู้สอนและสถาบันการศึกษาจะเป็นกุญแจสำคัญในการขยายใช้ระบบเหล่านี้ในวงกว้าง

### (2) Adaptive Learning: The Premise, Promise, and Pitfalls — Petr Johanes

ผู้วิจัย
 - Petr Johanes

หัวข้อเกี่ยวกับอะไร
 - บทวิเคราะห์ / วิจารณ์ (“premise, promise, and pitfalls”) ของ adaptive learning
 - มุ่งเน้นที่สิ่งที่ระบบ adaptive learning คาดหวังได้, โอกาสที่มี และปัญหาที่อาจเกิดขึ้นในการนำไปใช้จริง

ใช้เทคนิคอะไร
 - เป็นการวิเคราะห์ทางทฤษฎี (theoretical analysis), SWOT (อาจมีองค์ประกอบ) และสำรวจจากงานวิจัยต่าง ๆ ที่มีอยู่แล้ว (literature review) 
 - ไม่ได้ทดลองภาคปฏิบัติมากนัก แต่รวบรวมจุดแข็ง จุดอ่อน ความท้าทาย และเงื่อนไขที่ต้องการในการปรับใช้ระบบ adaptive learning อย่างได้ผล

ผลการวิจัย
 - Adaptive learning มีศักยภาพสูงที่จะช่วยให้การศึกษาเป็นส่วนตัว (personalized) มากขึ้น, รองรับผู้เรียนที่มีพื้นฐานแตกต่างกัน, เพิ่ม engagement และประสิทธิภาพของการเรียนรู้ 
 - แต่ก็มี pitfalls / ปัญหาหลายด้าน เช่น ข้อจำกัดด้านเทคโนโลยี, ความเป็นส่วนตัวของข้อมูล, ความยากของการออกแบบและบำรุงรักษาระบบ, บางครั้งผู้สอนไม่มั่นใจหรือไม่มีทักษะใช้, และการปรับตัวของผู้เรียนเองอาจมีอุปสรรค

ความคิดเห็นต่ออนาคต
 - จำเป็นต้องมีการวิจัยภาคปฏิบัติ (empirical studies) เพิ่มเติมเพื่อยืนยันว่า adaptive learning ทำงานได้ในบริบทที่หลากหลาย และรู้ว่าในเงื่อนไขใดได้ผลดีที่สุด 
 - การสำรวจเรื่องศีลธรรม (ethical issues), ความปลอดภัยและความเป็นส่วนตัวของข้อมูล, ความโปร่งใสของอัลกอริทึม เป็นเรื่องสำคัญในอนาคต 
 - อาจมีการพัฒนาระบบที่มี adaptivity สูงขึ้น ทั้ง real-time, มี feedback ที่ดี, สามารถปรับตามอารมณ์ /บริบทของผู้เรียน, และรองรับการใช้งานในวงกว้าง (scalability)
