## หัวข้อเกี่ยวกับอะไร
 - ศึกษาการออกแบบ open learner model (OLM) แบบเรียบง่าย (simple open learner models) สำหรับผู้เรียนที่มีแรงจูงใจสูง (intrinsically motivated, independent learners) โดยใช้ นักเรียนฝึกบิน (trainee pilots) เป็นกรณีศึกษา
 - สำรวจว่าผู้เรียนจะใช้โมเดลผู้เรียน (learner model) เพื่อวางแผนการเรียนของตัวเอง, ทำความเข้าใจจุดแข็งและจุดบกพร่อง, หรือใช้เป็นเครื่องมือวางแผนระยะยาวได้หรือไม่

## ใช้เทคนิคอะไร
 - นำระบบ Flight Club open learner model ไปใช้จริงที่โรงเรียนฝึกบินในสหราชอาณาจักร (UK) เป็นระยะเวลา 4 สัปดาห์
 - สังเกตพฤติกรรมผู้เรียนว่ามีการเข้าชม learner model, ใช้ข้อมูลจาก learner model เพื่อช่วยวางแผนการเรียน, หรือใช้เพื่อปรับกลยุทธ์การเรียนอย่างไร
 - มีการวิเคราะห์ว่าผู้เรียนใช้ “skill meters” หรือฟีเจอร์ใน learner model ใดบ่อยสุด, และดูว่าข้อมูล learner model ช่วยกระตุ้นการเรียนรู้หรือการวางแผนอย่างไร

## ผลการวิจัย
 - ผู้เรียนฝึกบินที่มีแรงจูงใจสูงจริง ๆ มักจะ เข้ามาดู learner model และใช้ข้อมูลจากนั้นเพื่อวางแผนการเรียนของตนเอง โดยเฉพาะเพื่อทำความเข้าใจระดับความรู้ปัจจุบัน จุดที่มีปัญหา และความเข้าใจผิด (misconceptions)
 - Learner model แบบเปิด (open) ถูกมองว่าเป็นประโยชน์สำหรับการวางแผนการเรียนระยะยาวและช่วยให้ผู้เรียนเห็นภาพของ “ที่ที่ตนอยู่” ในเส้นทางการเรียนรู้
 - อย่างไรก็ตาม ผู้เรียนบางส่วนไม่ได้ใช้ learner model อย่างต่อเนื่อง หรืออาจไม่เข้าดูในบางช่วงของการฝึก (มีอัตราการเข้าชม learner model แตกต่างกัน)

## ความคิดเห็นต่ออนาคต
 - Open learner models มีศักยภาพสูงในการ สร้างการสะท้อน (reflection) และช่วยให้ผู้เรียนสามารถเห็นภาพ “จุดที่ตนอยู่” และ “จุดที่ควรไป” ในการเรียนรู้ — ซึ่งอาจส่งเสริม metacognitive planning (การวางแผนโดยใช้การรู้จักรู้) ได้ดีขึ้น โดยเฉพาะในผู้เรียนที่มีแรงจูงใจสูง
 - อาจเป็นกุญแจสำคัญในการออกแบบระบบ ITS/Adaptive Learning ที่สนับสนุนการมีส่วนร่วมของผู้เรียน (learner agency) และช่วยให้ผู้เรียนติดตามความก้าวหน้าและปรับแผนการเรียนของตนเองได้อย่างอิสระ
 - แต่ก็มีคำถามว่าระบบแบบนี้จะเหมาะกับผู้เรียนที่ ไม่ได้มีแรงจูงใจสูงโดยกำเนิด หรือผู้เรียนทั่วไป (general learners) หรือไม่ — ซึ่งอาจต้องมีการออกแบบเพิ่มเติม เช่น การเตือนให้ดู learner model, การฝึกให้ผู้เรียนใช้ learner model, หรือการสนับสนุนจากผู้สอนเพื่อให้ learner model มีประโยชน์จริงในแง่ของการวางแผนการเรียนและการสะท้อนตัวเอง
