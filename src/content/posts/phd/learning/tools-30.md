## ผู้วิจัย
 - Chen, Saeedvand, Lai

## หัวข้อเกี่ยวกับอะไร
 - เสนอระบบ ALPN ที่ใช้ Knowledge Tracing + reinforcement learning เพื่อแนะนำเส้นทางเรียน (path) แบบ dynamic

## ใช้เทคนิคอะไร
 - โมเดล Attentive Knowledge Tracing (AKT) ประเมินสถานะความรู้, ใช้อัลกอริทึม Proximal Policy Optimization (PPO) เสริม entropy เพื่อเลือก path ที่หลากหลาย

## ผลการวิจัย
 - ระบบ ALPN ทำได้ดีกว่า baseline (learning gains + ความหลากหลายของ path) ประมาณ ~8.2% และ path หลากหลายขึ้น ~10.5%

## ความคิดเห็นต่ออนาคต
 - ผสาน ALPN กับ interface tutor / chat tutor เพื่อให้ path ที่ระบบเลือก “สอดคล้องกับสไตล์/ความสนใจ” ผู้เรียน
