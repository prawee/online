## หัวข้อเกี่ยวกับอะไร
 - การออกแบบ framework สำหรับ adaptive learning ในสถาบันอุดมศึกษา
 - เสนอแนวทางทางด้าน pedagogical (การสอน) และ system design (โครงสร้างระบบ) จากประสบการณ์จริงที่ University of Central Florida (UCF)

## ใช้เทคนิคอะไร
 - วิธี “field note” / case-study / descriptive: ใช้ข้อมูลจากการปฏิบัติงานจริงของอาจารย์ + instructional designers + administrators ที่ UCF ตั้งแต่เริ่มโครงการ adaptive learning (ปี 2014 เป็นต้นมา) 
 - กำหนด “5 key design features” ที่คิดว่าจำเป็นสำหรับ adaptive learning course (ทั้ง design, delivery) 
 - Best practices + lessons learned

## ผลการวิจัย
 - ได้รับ framework ที่ประกอบด้วย 5 ลักษณะสำคัญ (features) ที่ช่วยให้ adaptive learning course ถูกออกแบบและดำเนินการได้มีประสิทธิภาพขึ้น
 - ตัวอย่าง: การออกแบบที่ชัดเจนเรื่อง mastery learning, personalized content, feedback ที่เหมาะสม, การประเมินว่า adaptive learning คืออะไร (definition) ให้ทุกฝ่ายเข้าใจตรงกัน, การสนับสนุนจากสถาบันและบุคลากรที่เกี่ยวข้อง 
 - มีบทเรียนที่เรียนรู้ว่าการออกแบบ adaptive learning ต้องรวมทั้งมุม system + pedagogy, ต้องมีการฝึกอบรมผู้สอน, มีการสร้างความเข้าใจและแบ่งปันแนวปฏิบัติที่ดี (best practices) 


## ความคิดเห็นต่ออนาคต
 - adaptive learning design framework ที่ดีสามารถช่วยลดความไม่แน่นอนและเพิ่มโอกาสสำเร็จของโครงการ adaptive learning โดยเฉพาะในมหาวิทยาลัยที่เริ่มใช้ระบบนี้ 
 - ในอนาคตควรจะมีการวิจัยเพิ่มเติมเพื่อดูผลลัพธ์จริงในห้องเรียน (classroom outcomes), ระยะยาว (longitudinal studies), และขยายไปยังหลายสถาบันที่มีบริบทแตกต่างกัน (แตกต่างด้านทรัพยากร, เทคโนโลยี, ภาษา,วัฒนธรรม) 
 - การจัดเตรียมโครงสร้างพื้นฐาน, การสนับสนุนบุคลากร, และการยอมรับจากผู้เรียนและผู้สอน จะเป็นกุญแจสำคัญในการใช้งาน adaptive learning ให้ได้ผลดีในวงกว้าง
